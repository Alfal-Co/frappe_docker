Asset Import Template Guide

Columns:
- Asset Name: اسم الأصل (مثال: ثلاجة عرض رقم 001)
- Asset Category: اختر أحد التصنيفات (سيارات / ثلاجات العرض / مستودعات/مبانٍ / أصول أخرى)
- Warehouse: اختر مستودع التتبع (أحد المستودعات المعتمدة)
- Plate No: رقم لوحة السيارة (إلزامي للسيارات فقط)
- Model: الموديل (اختياري)
- Year: السنة (اختياري)
- Chassis No: رقم الشاصي (اختياري)
- Fridge Size: حجم الثلاجة (صغير / متوسط / كبير)
- Fridge Brand: علامة الثلاجة (اختياري)
- Branch: اسم الفرع (اختياري)

Examples:
سيارة:
Asset Name=سيارة توزيع 01, Asset Category=سيارات, Warehouse=سيارات - السليل - ALF, Plate No=ب ع ه 7959, Model=ايسوزو, Year=2025, Chassis No=JTE123456789, Fridge Size=, Fridge Brand=, Branch=السليل

ثلاجة:
Asset Name=ثلاجة عرض 197, Asset Category=ثلاجات العرض, Warehouse=ثلاجات - الدلم - ALF, Plate No=, Model=, Year=, Chassis No=, Fridge Size=كبير, Fridge Brand=الوطنية, Branch=الدلم

Note:
- Plate No إلزامي فقط عند إدخال سيارات.
